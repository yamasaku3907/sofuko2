インストール方法

1.sofukou2(GitHub)にアクセスする。その後<>Codeボタンを押す。

その後URLをコピーするか,zipファイルをダウンロードする。

2.任意のディレクトリに移動する。

$ cd {任意のディレクトリ}

3.1(URLをコピーした場合)以下のコードを実行する。

$ git clone https://github.com/yamasaku3907/sofuko2.git

3.2(zipファイルの場合)以下のコマンドを実行する。

$ unzip sofuko2-main.zip

4.展開したディレクトリに移動する。

$ cd sofuko2-main

5.中身を確認する。

$ ls -l

total 8

README.md
